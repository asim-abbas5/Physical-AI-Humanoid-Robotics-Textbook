# Welcome to Physical AI & Humanoid Robotics

Welcome to the **Physical AI & Humanoid Robotics** textbook - an interactive, AI-native guide to building intelligent robotic systems.

## What You'll Learn

This textbook covers the complete stack for developing humanoid robots and embodied AI systems:

### Module 1: The Robotic Nervous System (ROS 2)
Learn the fundamentals of ROS 2, the industry-standard framework for robot software architecture.

### Module 2: Simulated Twins & Embodied Training
Master digital twin simulation using Gazebo and learn how to train robots in virtual environments.

### Module 3: NVIDIA Isaac Platform
Explore NVIDIA's Isaac Sim and Isaac SDK for physics-accurate simulation and AI-powered robotics.

### Module 4: Vision-Language-Action Models (VLA)
Discover how modern AI models enable robots to understand commands and interact with the physical world.

## How to Use This Textbook

- **Interactive Learning**: Select any text and click "Ask AI" to get context-aware explanations
- **Hands-On Examples**: All code examples are tested and runnable
- **Progressive Complexity**: Modules build on each other, starting from fundamentals
- **Authoritative Sources**: Every claim is cited from official documentation and research

## Prerequisites

- Basic Python programming
- Familiarity with Linux command line
- Understanding of basic robotics concepts (helpful but not required)
